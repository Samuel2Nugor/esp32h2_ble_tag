# Hardware overview

## Main hardware

### ESP32-H2-DevKitM-1
Main development board used for the BLE tag firmware.

Official reference:
- [ESP32-H2-DevKitM-1 documentation](https://docs.espressif.com/projects/esp-dev-kits/en/latest/esp32h2/esp32-h2-devkitm-1/user_guide.html)

### 2.13" e-paper module by WeAct Studio
Display used to show the selected payload fields.

Reference:
- [WeAct Studio Epaper Module](https://github.com/WeActStudio/WeActStudio.EpaperModule)

### USB cable
Used for power, flashing, and serial monitoring.

### Phone with nRF Connect
Used to scan, connect, and test the BLE tag.

References:
- Android: Google Play
- iPhone (iOS): App Store
